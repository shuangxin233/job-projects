# 食谱知识库问答与推荐系统 - 部署说明

## 项目定位

基于本地 Markdown 食谱语料的轻量级问答与推荐系统。项目主线在 `job_project/app.py`，会读取 `data/C8/cook` 下的 Markdown 菜谱，进行标题抽取、规则路由、关键词打分和证据片段输出。

## 跨设备运行结论

可以直接在其他设备运行。该项目不依赖大模型 API，也不需要向量数据库；只需要 Python。

## 启动方式

```powershell
cd 食谱知识库问答与推荐系统
python job_project\app.py --query "宫保鸡丁怎么做"
python job_project\app.py --query "鸡肉和蔬菜能搭配做什么" --top-k 5
```

## 目录说明

```text
job_project/app.py      求职项目主入口
data/C8/cook            运行所需 Markdown 菜谱语料
JOB_PROJECT_MAINLINE.md 面试展示主线说明
```

## 注意事项

- 本目录只保留求职演示需要的主线代码和 Markdown 菜谱语料。
- 原 all-in-rag 教程仓库体积较大，本次没有整仓上传。
- 当前实现是轻量检索增强问答，不是完整向量库 RAG；可后续升级 embedding、ChromaDB、Top-K 语义检索和 LLM 生成。
