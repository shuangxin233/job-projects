# 食谱知识库问答与推荐系统 - 部署说明

## 项目定位

这是一个基于本地 Markdown 食谱语料的轻量级问答与推荐系统。主程序位于 `job_project/app.py`，会读取 `data/C8/cook` 下的菜谱文档，并完成标题抽取、问题路由、关键词打分和证据片段输出。

## 跨设备运行结论

可以直接在其他设备运行。当前版本不依赖大模型 API，也不需要数据库服务，只需要 Python。

## 启动方式

```powershell
cd 求职项目\食谱知识库问答与推荐系统
python job_project\app.py --query "宫保鸡丁怎么做" --top-k 3
python job_project\app.py --query "鸡肉和蔬菜能搭配做什么" --top-k 5
```

## 目录说明

```text
job_project/app.py      项目主入口
data/C8/cook            运行所需 Markdown 食谱语料
JOB_PROJECT_MAINLINE.md 项目主线说明
README.md               项目总说明
```

## 注意事项

- 当前实现是轻量检索增强问答，重点展示知识库检索、问题路由和 Top-K 证据召回。
- 如需升级为完整 RAG，可继续加入 embedding、ChromaDB、语义检索和 LLM 生成。
- 新设备只要保留 `job_project` 和 `data/C8/cook`，即可复现命令行检索流程。
